### cupp

https://github.com/Mebus/cupp

Permet la création de wordlist de manière interactive.

```bash
cupp -i
```

### Username anarchy

https://github.com/urbanadventurer/username-anarchy Permet de créer des permutations sur des noms d'utilisateurs.

```bash
./username-anarchy Bill Gates > bill.txt
```

### Cewl

Cewl va parser des pages web et créer une wordlist à partir des termes les plus pertinents.

```
cewl domain.com -m 5 -w wordlist.txt
```
